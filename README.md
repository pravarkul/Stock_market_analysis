# Stock Market Analytics Dashboard

## Project Overview

An end to end stock market analytics platform built using Python, MySQL and Power BI.

The system automatically downloads stock market data from Yahoo Finance, stores it in MySQL, performs technical indicator calculations and visualizes insights through an interactive Power BI dashboard.

---

## Tech Stack

Python,
MySQL,
Power BI,
Yahoo Finance API,
Pandas,
SQLAlchemy

---

## Features

### Automated Data Pipeline

Daily stock data collection using Yahoo Finance.

### Incremental Updates

Only new records are added to the database.

### Technical Indicators

MA20,
MA50,
RSI,
Volatility

### Stock Analysis

Risk vs Return,
Performance Rankings,
Volume Analysis,

### Sector Analysis

Sector Comparison,
Sector Risk Analysis,
Sector Performance Ranking
